Role of Level Users

  • Admin creates LevelUsers and assigns level users as State/Division/District/Block Grievance access level.
  • Based on their Level and Grievance Access level, level user can see grievances assigned to him on his/her dashboard.
  • Level user has access to close the Grievance or to forward the grievance to next level user.
  • Level Users can only close the Grievance,who has access to Close the Grievance, which is assigned by admin.
  • If Level Users wants to know the progress click on view button,which will show Timeline from GrievanceRegistered to Grievance closed.

How to close the Grievance?

  • level user can see Grievances which are assigned to him under Accepted Grievances.
  • if user wants to close grievance,click on More Details
    1. provide text in mandatory comment about the Grievance
    2. select Do you think the grievance is redressed? Radio button to yes
    3. select Resolved in Redressed Status dropdown and submit.
      4. Selected Grievance complaint will be closed.
